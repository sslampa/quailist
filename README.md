# quailist
List making app for quails

# Start Server
`mongod` -> Separate tab

`npm start` -> Starts dev server at `localhost:3000`

# Run Tests
`npm test`

# File Organization

```
server/
  controllers/
  models/
client/
  public/
  components/
tests/
  client/
  server/
index.html
server.js
```
